# Proyecto

Proyecto scaffold generado con Vite + React + TypeScript.

Requisitos: Node.js (v18+ recomendado) y npm.

Instrucciones (PowerShell):

```powershell
cd 'C:\Users\User\OneDrive\Escritorio\proyecto'
npm install
npm run dev
```

Luego abre la URL que muestre Vite en la terminal (normalmente http://localhost:5173).
